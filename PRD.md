📋 [Interim PRD] 세원뮤직 AI 비서 구축 프로젝트 블루프린트
1. 프로젝트 개요
프로젝트명: 세원뮤직 전용 AI 업무 관리 비서

운영 환경: Mac 환경 (로컬 스크립트 및 파일 시스템 중심 구축)

최종 목표: 온·오프라인 매장 병행에 따른 과도한 업무(거래처 소통, 정산, 재고, 광고 관리)를 AI를 통해 자동화 및 효율화.

2. 현재 진행 상황 및 개발 범위 (Phase 1 MVP)
현재 가장 시급한 '1번 기능(거래처 대화 및 파일 정리, 일정 트래킹)'의 아키텍처 수립 완료.

[Phase 1 업무 파이프라인]
데이터 수집 (Ingestion): * 카카오톡 GUI 자동화 불가로 인하여, 거래처 대화록(.csv 또는 .txt)과 공유 파일, 미디어(이미지/동영상)를 사용자가 로컬의 임시 다운로드 폴더(`[00_수동다운로드_RAW]`)에 수동으로 백업.

전처리 자동화 (Preprocessing):
Python 스크립트를 통해 임시 다운로드 폴더를 감시 및 스캔하여, 대화록과 파일들을 거래처별/날짜별로 자동 분류 및 `[01_정리전_대화및파일]` 구조로 이동.

AI 분석 및 아카이빙 (Processing):

Python 스크립트(pandas 활용)로 카카오톡 CSV 포맷을 읽어 파싱.

LLM API를 연동하여 대화 맥락 요약, 공유 파일 매칭, 날짜 기반 일정 추출 수행.

결과물을 검색이 용이한 YAML Front Matter를 포함한 마크다운(.md) 문서로 자동 생성하여 [02_정리완료_업무일지] 폴더에 저장.

액션 플랜 연동 (Action):

추출된 일정을 Mac의 '미리 알림(Reminders)' 또는 '캘린더' API와 연동하여 자동 등록.

3. 검토된 데이터 샘플 구조
거래처: 데임악기 (김영준, 조성찬)

대화 특징: 위탁 배송(직발송) 요청 정보(고객 주소, 연락처), 송장 번호 공유, 신제품 입고 리스트 파일(SIRE V NewGen(가격포함).xlsx) 및 재고 문의 파일(Sire_Guitar_Models.xlsx) 교환이 빈번함.

4. Next Action (새 세션에서 시작할 작업)
수동 다운로드한 카카오톡 파일들을 감지하여 거래처/날짜별로 분류하고, 대화록을 파싱하여 링크 메타데이터를 추출하는 전처리 자동화 Python 스크립트 작성 및 테스트.

전처리된 데이터를 바탕으로 LLM API(Claude/OpenAI 등)를 연동하여 마크다운(.md) 파일 형태의 업무 일지를 자동 생성하는 파이프라인 구축.

2번 기능(재고 관리) 확장을 위해 대화 속 엑셀 파일 데이터를 AI 비서와 어떻게 연동할지 구체화.
